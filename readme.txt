== Description ==

This plugin allows you to fetch data from your Nyx account and display upcoming ticketed events on your Wordpress website.

== Frequently Asked Questions ==

= How to get started? =

Fetch your NYX API Key from your nyx customer account. After, head back to your WordPress and hit "Nyx settings" and activate your integration.

= How do i show a ticket view? =

Once created, you can include it on any page using the shortcode provided on the ticket view list.
